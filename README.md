# ほうどう寺サイトプロジェクト2025

ほうどう寺のWEBサイト2025年版です。

## 技術選定

- Svelte, SvelteKit
- TypeScript
- ESLint
- StyleLint
- スタイル：CSS

## 開発にかかわるには

ローカルにリポジトリをクローンし、

```shell
pnpm install
pnpm dev --open
```

フレームワークについては[Svelte • Web development for the rest of us](https://svelte.jp/)を見てください。
